============================================================Installing Python on macOS============================================================Different versions of Python for macOS can be found here:https://www.python.org/downloads/macos/Anaconda distribution of Python for macOS can be found here:https://www.anaconda.com/products/individualThe installation instructions for Anaconda distribution are available here:https://docs.anaconda.com/anaconda/install/mac-os/Below we provide instructions to help users install a recommended Python versionsuitable for NRAP-Open-IAM.1. Run "python-3.9.7-macosx10.9.pkg" file in the folder "installers/macOS"	-- During installation, select "Add Python to PATH".	-- Leave all other settings to the default selections (unless doing unique	Python installations, for experienced users))2. Run "Install_Extensions.sh" file in the folder "installers/macOS"	-- This will download the required dependencies to run the application.3. Installation is complete.We provide additional instructions to help more experienced users (familiar with virtual environments)create a Python virtual environment suitable for NRAP-Open-IAM in the file "Virtual_Python_Environment.txt".============================================================General requirements for Python Distributions on macOS============================================================Needed software:Python 3.9 or above (we recommend Python 3.9.7)gfortranNeeded Python libraries:pipnumpyscipypandasmatplotlibPmwcertifipyDOEpytzPyYAMLtornadojupytertensorflowkerasscikit-learn============================================================Testing Python and NRAP-Open-IAM Setup============================================================After installing Python and required packages please test the Python installationand setup of NRAP-Open-IAM by running "openiam_setup_tests.py" in the folder"setup" of the NRAP-Open-IAM root folder.After testing the installations return to the root directoryof the NRAP-Open-IAM distribution and run the GUI by selecting/running the"Run_OpenIAM_GUI_macOS.sh" file, or navigate directly to the folder "source/GUI"and run file "NRAP_OpenIAM.py".
